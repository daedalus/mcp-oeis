# MCP OEIS

MCP server for the OEIS (Online Encyclopedia of Integer Sequences) API.

## When to use this skill

Use this skill when you need to:
- Look up integer sequences
- Search sequences by terms or name
- Get information about mathematical sequences

## Tools

- `get_sequence_by_id` - Get sequence by OEIS ID (e.g., A000045 for Fibonacci)
- `search_by_terms` - Search by providing integer terms
- `search_by_name` - Search by name or keyword

## Install

```bash
pip install mcp-oeis
```
